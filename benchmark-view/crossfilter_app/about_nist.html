This site is hosted and managed by the Hennig group at the University of Florida and is a result of a collaboration between
NIST and the Hennig group at the University of Florida and has been partially funded through the
Materials Genome Initiative for DFT Benchmarking.
All content is available for public use. If you use our data in your research, we ask that you cite MPInterfaces which was used in the high throughput generation of the data.

  • The VASP data in this site was computed at UF's supercomputing facitilies of HiperGator
    using MPInterfaces which is a high throughput Python framework developed
    by the Hennig group for the setup, computation and analysis of materials interfaces and 2D materials
  • The Dmol3 and Quantum Espresso data have been computed at NIST using in house high throughput capabilities
  • The statistical tools implemented in this website are apps developed using the
    Shiny by R studio which is a web framework that makes use of the powerful R language specialized in data analysis in computing.
  • The user interaction with the database to crossfilter the data based on the user interface is based on a REST API
    run through the Python Flask app DFT Crossfilter which is under development
    that interacts with a secure MongoDB database hosted by the Hennig group
  • This work was supported by NIST under grant 00095176 and by NSF under the grants DMR-1542776 and ACI-1440547
