DFT Benchmarking Data Analytics

This web platform allows the user to benchmark DFT calculations for basic material properties using two codes (VASP and DMol3). This web platform is the result of an MGI–sponsored (Materials Genome Initiative) collaboration between NIST (Materials Science and Engineering Division) and the University of Florida.

  • Hover over elements in red in the periodic table to view the Volume per atom (A^3), Bulk Modulus (GPa) and its pressure derivative estimated at an infinite k-points density by the Pade extrapolation, each in their fcc, bcc and hcp phases. This data is calculated with VASP using the PBE exchange correlation functional
  • · Use the drop down menus below the Periodic Table to visualize the raw data (energy, Volume/atom, Bulk Modulus (B) and dB/dP) for whichever code (VASP or DMol3), structure (fcc or bcc or hcp) and exchange-correlation (PBE or LDA) of interest.
